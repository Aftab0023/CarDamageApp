# 🚗 Car Damage Detection System

---

## 🟢 1. Introduction
This project is an AI-based system that detects damage in a car image.  

👉 The user uploads an image of a vehicle  
👉 The system analyzes it using Artificial Intelligence  
👉 It shows:  
- Damaged parts  
- Damage percentage  
- Severity level  
- Estimated repair cost  
- Insurance suggestion  

---

## 🟢 2. Objective
The main goal of this project is:  
- To automatically detect car damage  
- To reduce manual inspection effort  
- To help in insurance claim decisions  

---

## 🟢 3. Technologies Used
- **Python** → Main programming language  
- **Flask** → Backend web framework  
- **YOLO (AI Model)** → Object detection  
- **OpenCV** → Image processing  
- **HTML + CSS** → User interface  

---

## 🟢 4. How the System Works (Step-by-Step)
👉 **Step 1:** User uploads car image  
👉 **Step 2:** Image is sent to Flask backend  
👉 **Step 3:** YOLO model processes image  
- Detects damaged areas  
- Draws bounding boxes  

👉 **Step 4:** System calculates:  
- Damage percentage  
- Damaged parts  

👉 **Step 5:** Additional features:  
- Severity (Low / Medium / High)  
- Repair cost estimation  
- Insurance advice  

👉 **Step 6:** Result is displayed on screen  

---

## 🟢 5. Output Example
The system shows:  
- **Damage Rate:** 39%  
- **Damaged Parts:** Door dent  
- **Severity:** High  
- **Estimated Cost:** ₹5000  
- **Advice:** Recommended to claim insurance
